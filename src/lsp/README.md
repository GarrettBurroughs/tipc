## Features

The language server protocol is designed such that a language server can support a subset of all the available features enabled by the protocol.
Currently, the TIP language server supports:

[x] Parsing Errors
[x] Type information
[ ] Code Formatting
[ ] Go to definition
[ ] Autocompletion
